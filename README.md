## Install + run

```bash
# download the app
# macOS: .dmg from releases
# windows: .exe from releases
```

[→ Download latest release](https://github.com/heynaavi/Whip-Me-Bad/releases/latest)

Or run from source:

```bash
git clone https://github.com/heynaavi/Whip-Me-Bad.git
cd Whip-Me-Bad
npm install
npm start
```

## How it works

- **Press Enter anywhere.** Get whipped. 🍑
- **Use Kiro?** Hooks auto-install. Whips on prompts, edits, tasks, agent stop.
- **Mash Enter.** Phrases escalate. It gets... intense.
- **Menu bar app.** Pause, volume, custom sounds, custom hotkey.
- **Insights.** Track your whips. Earn a persona.

## Contribute

The app listens on `localhost:31338/whip`. If your IDE can run a command on an event, it works.

| IDE | Status |
|-----|--------|
| Kiro | ✅ Auto-installed hooks |
| Cursor | 🔲 Needs hook system — PRs welcome |
| Windsurf | 🔲 Same |
| VS Code | 🔲 Could be an extension |

Other ways to help: new animations, sound packs, Windows key monitoring, Linux support.
